# Final Deadline Timer

## Project Description

這是一個全用 vibe coding 生出來的小型垃圾，目的是為了提醒我們可愛的社員們記得交專案

## Features

- 精度到秒數的倒數計時
- 外觀抄襲(?)星際大戰
- 秒數變動有動畫
- 背景有閃閃的星星
- 每一分鐘會有一個流星流過去
- 給社員壓力
